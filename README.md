Movie Recommendation System – Collaborative Filtering

Overview

This project implements a Movie Recommendation System using the MovieLens dataset. It demonstrates User-Based and Item-Based Collaborative Filtering approaches to recommend movies based on user ratings. Evaluation metrics include RMSE, MAE, and Precision@K, and key results are visualized using heatmaps and charts.

Scenario 1 – User-Based Collaborative Filtering
  
In User-Based CF, movies are recommended to users based on the preferences of similar users. A User-Item matrix is created, missing ratings are filled with 0, and user similarity is computed using Pearson correlation. Top-N similar users are identified to predict ratings for unseen movies. This approach achieved RMSE = 0.967 and MAE = 0.75, showing that recommendations are reasonably accurate.

Scenario 2 – Item-Based Collaborative Filtering

   In Item-Based CF, similar movies are recommended based on a user’s past ratings. An Item-User matrix is created, missing values are filled with 0, and item similarity is computed using cosine similarity. Recommendations are generated by combining user ratings with item similarity scores. This approach achieved RMSE = 0.966 and Precision@5 = 0.8, indicating good recommendation quality and relevance.

Analysis

  Item-Based CF performed slightly better in precision, making it more effective for recommending relevant movies. Both methods provide low RMSE and MAE, confirming the models’ predictive accuracy. Heatmaps and charts help visualize the user-item interactions and similarity relationships between users or items.
